# Installation

#### Using Air

To add Air via `go install` (Recommended) with Go 1.23 or higher:

```
go install github.com/air-verse/air@latest
```

```
nano ~/.zshrc
export PATH=$PATH:/Users/qaiswardag/go/bin
```

Run

```
air -c .air.toml
```
